# Ithi's Personal Portfolio

Personal **portfolio website** (work in progress) built with HTML, CSS, and JavaScript.   
Live demo (open locally): just open `index.html` in your browser after cloning the repo.

## Features

- Home/landing page (`index.html`) with introduction.
- Separate blog listing page (`blogs.html`) from my Medium page.
- Shared styles and scripts organised into folders (`styles/`, `scripts/`, `files/`). 

## Project Structure

├─ files/  
├─ scripts/  
├─ styles/  
├─ .gitignore   
├─ README.md  
├─ blogs.html  
└─ index.html 
 
## Getting Started

Clone the repository and open it locally:

```
git clone https://github.com/lostithi/bug-free-portfolio.git
cd bug-free-portfolio
```
 
Then open `index.html` (and `blogs.html` if you want) in a browser. 

## Development

- Edit `index.html` and `blogs.html` to change content.
- Update styles in `styles/` (e.g. main layout, colours, typography).
- Add images or other assets into `files/` and reference them from your HTML.
